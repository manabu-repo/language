## Korean alphabet

| 字母 | 发音 | 拼音  |
| ---- | ---- | ----- |
| ㅗ   | /o/  | or    |
| ㅐ   | /ae/ | ei    |
| ㅅ   | /s/  | si    |
| ㅂ   | /b/  | bu/pu |

## Combination

> ㅅ + ㅗ = 소

**소** /so/ so

> ㅅ + ㅔ = 새

**새** /sae/ sei

> ㅂ + ㅗ = 보

**보** /bo/ bu

> ㅂ + ㅐ = 배

**배** /bae/ pei
