## Korean alphabet

| 字母 | 发音 | 拼音 |
| ---- | ---- | ---- |
| ㅏ   | /a/  | a    |
| ㅓ   | /eo/ | o    |
| ㄱ   | /g/  | ka   |
| ㅁ   | /m/  | ma   |

## Combination

> ㄱ + ㅏ = 가

**가** /ga/ ka

> ㄱ + ㅓ = 거

**거** /geo/ geo

> ㅁ + ㅏ = 마

**마** /ma/ ma

> ㅁ + ㅓ = 머

**머** /meo/ meo
